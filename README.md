BBS
========
>>这是一个界面美观，功能基本完整的论坛。
---
### 可以直接使用，将项目导入eclipse，然后再把sql导入MySQL数据库就OK了。
---
### 技术栈
**jsp** **servlet** **javabean** **mysql** **html** **css** **javascript** **jQuery**
